# SDMX Discovery

> A beginner-friendly guide to exploring SDMX APIs, metadata, and real-world analytics with Python.

## Overview

SDMX Discovery is a hands-on learning project that helps data scientists, analysts, economists, researchers, and developers understand and work with Statistical Data and Metadata eXchange (SDMX).

The goal of this project is to bridge the gap between the SDMX standard and practical data analytics by providing simple, reproducible examples using real-world datasets. Through notebooks, visualizations, and exploratory analyses, this repository demonstrates how to discover, query, transform, and analyze SDMX data using Python.

Whether you are completely new to SDMX or looking to deepen your understanding of the ecosystem, this project provides a structured path from your first API call to meaningful analytical insights.

## Why This Project?

SDMX powers data exchange across many international organizations, central banks, national statistical offices, and financial institutions. Despite its widespread adoption, getting started can be challenging due to the complexity of concepts such as:

* Dataflows
* Data Structures (DSDs)
* Dimensions
* Codelists
* Metadata
* API Querying

This repository aims to make these concepts easier to understand through practical examples.

## Project Objectives

* Learn the fundamentals of the SDMX standard
* Explore SDMX APIs from various data providers
* Understand SDMX metadata structures
* Build reusable Python utilities for SDMX data extraction
* Create visualizations and analytical workflows
* Develop real-world use cases using economic and financial datasets
* Document lessons learned for future SDMX practitioners
